### Can Claude Find Fraud?

[This is a good one](https://www.dolthub.com/blog/2026-02-26-claude-find-fraud/). 

I imported the newly released [Medicaid payments database](https://www.dolthub.com/repositories/timsehn/medicaid_provider_spending/data/main) into Dolt. I fired up the [Dolt Workbench's agent mode](https://www.dolthub.com/blog/2026-02-09-introducing-agent-mode/) and asked the agent to identify suspicious payment patterns.

[The results are fascinating](https://www.dolthub.com/blog/2026-02-26-claude-find-fraud/). Using an agent to do data analysis alongside a SQL workbench produced amazing results in record time. I have more to come here.
