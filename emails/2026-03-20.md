The team has been hard at work [improving agent mode](https://www.dolthub.com/blog/2026-03-19-agent-mode-features/) in Dolt Workbench. ### SSH Remotes Are Here

[You can now push and pull Dolt databases using SSH](https://www.dolthub.com/blog/2026-03-17-ssh-remotes/). This was a popular request from teams who already have SSH infrastructure in place. Neil wrote up the announcement and explained how you can leverage your existing SSH key infrastructure for secure database operations. If you're already using SSH-based Git workflows, you'll feel right at home with Dolt SSH remotes.

### lazydolt: Git's Favorite TUI Comes to Dolt

If you've used [lazygit](https://github.com/jesseduffield/lazygit), you'll love [lazydolt](https://www.dolthub.com/blog/2026-03-16-lazydolt/).
