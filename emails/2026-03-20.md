### SSH Remotes

[You can now push and pull Dolt databases using SSH](https://www.dolthub.com/blog/2026-03-17-ssh-remotes/). This was a long standing request from folks who already have SSH infrastructure in place. If you're already using SSH-based Git workflows, you'll feel right at home with Dolt SSH remotes.

### lazydolt

If you've used [lazygit](https://github.com/jesseduffield/lazygit), you'll love [lazydolt](https://www.dolthub.com/blog/2026-03-16-lazydolt/). It's a Terminal User Interface built in Go that lets you navigate branches, view diffs, manage commits, and run database operations without leaving your terminal. The interesting bit is that it was built by coding agents. Things that were not worth building a year ago, are suddenly worth building.
